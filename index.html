Blog常用指令-Hexo篇

Hexo：

  1. hexo init [folder]
    在当前目录初始化一个名为folder的博客项目；
  2. hexo new [layout] <title>
    创建一篇新文章
    layout：可指定布局，默认为post(文章)，可选draft(草稿)、page(页面)。
  3. hexo generate / hexo g
    将源文件生成静态文件(默认输出到public文件夹)
  4. hexo server / hexo s
    启动本地服务器，默认访问地址 http://localhost:4000
    -p <port>
    指定端口 (例如hexo s -p 4001，可解决端口被占用问题)
  5. hexo deploy / hexo d
    根据_config.yml中的deploy配置，将生成的静态文件部署到网络(上传)
  6. hexo clean
    清除缓存文件(db.json)和已生成的静态文件(public文件夹)
  7. 组合指令
    如hexo g;hexo s;hexo d
    先生成静态文件，再启动本地服务器，再上传部署。
